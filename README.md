## Key Components

- **Image**
  - Wait until in-view
  - Wait until loaded
  - Play animation on load
  - Show placeholder
- **Image Carousel**
  - Timer before change
  - Pause timer on hover
- **Components**
  - Parallax
    - Accordian
    - On-scroll reveal
    - Mouse/ Scroll Interactions
    - Before/ After Sliders
    - Image Zoom
- **Loaders**
  - Page waits for assets to load before showing
  - Graceful image reveals
